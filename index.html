Type of SQL Commands

  1. DDL: Data Definition Lenguage

    2. Can be use to define the database squema this commands are use to create, modify and delete database structures

    • CREATE
    • DROP
    • ALTER
    • TRUNCATE
    • COMMENT
    • RENAME
  2. DML: Data Manipulation Lenguage

    3. It deals with the manipulation of the data present in an specific database

    • INSERT
    • UPDATE
    • DELETE
    • LOCK
  3. DCL: Data Control Lenguage

    4. This commands deal with the permissions fro the data in certain databases

    • GRANT
    • REVOKE
  4. TCL: Transaction Control Lenguage

    5. Assings task to the database, starting with specific conditions, and ending up when the conditions set are met, allowing the transaction to be completed

    • COMMIT
    • ROLLBACK
    • SAVEPOINT
    • SET TRANSACTION
  5. DQL: Data Query Lenguage

    6. Can be use to get the data out of the database and imposing a commant to it

    • SELECT
